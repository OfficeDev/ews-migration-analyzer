## EWS Code Analyzer Diagnostics
| Diagnostic ID | Severity | Meaning |
|---|---|---|
| EWS001 | Error | Graph API equivalent is generally available |
| EWS002 | Warning | Graph API equivalent is in preview |
| EWS003 | Warning | Graph API equivalent not yet available |
| EWS000 | Warning | Unmapped operation |
| EWS004 | Info | Summary count of EWS references |
| EWS005 | Warning | Call to action with migration readiness percentage |
